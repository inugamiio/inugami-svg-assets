**inugami-svg-assets** is a modern Angular component library designed for the Inugami project ecosystem.
It provides high-performance, reusable, and modular UI icons.

---

## 🚀 Features

* **Angular Ready**: Built with the latest Angular standards.
* **Lightweight**: Minimal footprint and dependencies.

```bash
